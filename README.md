# Project Description
This project demonstrates hands-on SQL data analysis skills using the Northwind sample database.
It includes structured SQL queries written to answer real-world business questions related to sales, customers, orders, products, and revenue insights.

# Project Objective
Practice and demonstrate SQL querying skills
Extract meaningful business insights from relational data

Key Tables: Customers, Orders, Order_Details, Products, Employees, Suppliers, Categories
Use Case: Sales performance, customer behavior, product analysis, revenue trends
